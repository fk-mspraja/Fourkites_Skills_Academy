## Product Catalog

### 1. Intelligent Control Tower (ICT)

**Vision:** "From the speed of consensus to the velocity of autonomy"

**Core Capabilities:**

#### Real-Time Visibility
- **Shipment Tracking:** 3M+ shipments tracked daily
- **Carrier Network:** 500,000+ carriers integrated
- **Transportation Modes:** Ground (LTL/FTL), Ocean, Air, Rail, Parcel
- **GPS Sources:** ELD, Mobile apps, Carrier APIs, Telematics providers
- **Update Frequency:** Every 5-15 minutes

**Value Proposition:**
- Eliminate manual check-calling
- Single view across all carriers and modes
- Proactive exception management
- Reduce transportation costs 5-10%

#### Exception Management
- **Automatic Detection:** Delays, route deviations, missed appointments
- **Predictive Alerts:** ETA changes before customer impact
- **Smart Notifications:** Role-based alerts via email, SMS, Slack
- **Resolution Workflows:** Automated escalation and remediation

**Value Proposition:**
- Reduce customer complaints 30-50%
- Improve on-time delivery 10-20%
- Decrease detention/demurrage costs

#### Analytics & Reporting
- **Pre-Built Dashboards:** Carrier performance, lane analytics, exception trends
- **Custom Reports:** Drag-and-drop report builder
- **Data Export:** API access to all tracking data
- **BI Integration:** Power BI, Tableau, Looker connectors

**Value Proposition:**
- Data-driven carrier selection
- Identify cost-saving opportunities
- Measure logistics KPIs

### 2. Digital Workers

**Vision:** Autonomous operations that replace coordination-based workflows

**Available Workers:**

#### 1. Appointment Scheduler
- **Function:** Automatically schedule dock appointments
- **Integration:** TMS, WMS, carrier portals
- **Impact:** 80% reduction in manual scheduling time

#### 2. Load Tender Agent
- **Function:** Automated load tendering and carrier assignment
- **Logic:** Rate optimization, carrier performance, capacity
- **Impact:** 50% faster tender cycle, 10-15% cost savings

#### 3. Track & Trace Agent
- **Function:** Autonomous shipment monitoring and customer updates
- **Channels:** Email, SMS, customer portals
- **Impact:** Zero manual status requests

#### 4. Exception Handler
- **Function:** Auto-resolve common exceptions (e.g., ETA delays)
- **Actions:** Notify stakeholders, reschedule appointments, update forecasts
- **Impact:** 60% of exceptions resolved without human intervention

#### 5. Document Processing Agent
- **Function:** Extract data from BOLs, PODs, invoices
- **Technology:** Computer vision + LLMs
- **Impact:** 95% accuracy, 90% time savings

**Pricing Model:** Per-transaction or subscription-based

### 3. Integrations & APIs

#### TMS Integrations
- **Supported Systems:** Oracle TMS, SAP TM, Blue Yonder, MercuryGate, Manhattan
- **Sync:** Bi-directional (load creation, status updates, POD upload)

#### ERP Integrations
- **Supported Systems:** SAP, Oracle, Microsoft Dynamics
- **Use Cases:** Order-to-cash visibility, inventory optimization

#### Carrier Integrations
- **Methods:** API, EDI, Portal scraping (JustTransform for ocean)
- **Coverage:** 500K+ carriers globally

#### Custom API
- **REST API:** Full access to tracking data, analytics, configuration
- **Webhooks:** Real-time push notifications for events
- **Documentation:** Developer portal with SDKs (Python, Java, Node.js)

### 4. Ocean Visibility

**Capabilities:**
- **Container Tracking:** 150+ shipping lines via JustTransform
- **Port Visibility:** ETAs, vessel schedules, port congestion
- **Multimodal:** Door-to-door visibility (ocean + drayage)

**Value Proposition:**
- Reduce demurrage/detention 20-30%
- Optimize container pickup timing
- Improve inventory planning accuracy

### 5. Yard Management (Emerging)

**Capabilities:**
- **Dock Scheduling:** Appointment management
- **Gate Check-In/Out:** Driver self-service
- **Asset Tracking:** Trailers, containers in yard

**Status:** Beta (select customers)
